# todo-app
fullstack ToDo app

## Backend

- express
- mongoDB: mongoose
- dotenv

- dev:
	- nodemon


## Frontend

- vite
- react

- react-router
- sonner (for Toaster pop-up notification)
- axios	(API send-receive)
- lucide-react (icon)

- tailwindcss
- shadcn (lib of tailwindcss components): need to use jsconfig.json instead of tsconfig

- background: https://patterncraft.fun/
